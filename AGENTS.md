# Project Documentation: Order System and Door Configurator

This document provides a comprehensive overview of the Order System and Door Configurator project

## 1. Project Overview

This project is two main things:
- **Door/Opening Configurator**: This will be a canvas on a webpage that will allow a user to configure an opening of a building. This will contain framing, doors, and hardware.
- **Order System**: This will be like a shopping cart. The user can finish configuring an opening, place them into a cart, and a quote can be created from this cart of openings.

## 2. Tech Stack

- **Backend**: Node.js, JavaScript/TypeScript
- **Frameworks**: Three.js, Vite
- **Package Manager**: npm

## 3. Project Structure

The project is organized into the following directoryies:

- **src**: Contains all the code to run in the webpage

## 4. Core Components

- **index.html**: Contains the main webpage to display.
- **main2.ts**: Contains the main TypeScript page to place all the functionality

## 5. Program Flow

This is the general flow for a user in this program:

1. They will get a dashboard showing all of their open quotes
2. They will have a button to configure a new door/opening
3. They will create their door/frame/opening as they desire
4. They will add this opening to a cart
5. When they have completed adding all of their openings, they can submit the quote.
6. After quotes are submitted, they can appear on their dashboard.
